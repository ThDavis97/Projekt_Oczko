Deletes a message buffer that was previously created using a call to \doxyref{x\+Message\+Buffer\+Create()}{p.}{message__buffer_8h_a2959cd0e3d2bd20d46908e5c9872be36} or \doxyref{x\+Message\+Buffer\+Create\+Static()}{p.}{message__buffer_8h_acc84f4a15dbd39ecbad35f7337e78b2c}. If the message buffer was created using dynamic memory (that is, by \doxyref{x\+Message\+Buffer\+Create()}{p.}{message__buffer_8h_a2959cd0e3d2bd20d46908e5c9872be36}), then the allocated memory is freed.

A message buffer handle must not be used after the message buffer has been deleted.
